# LibMedia
 media的引用


#### MediaFactory
media的工厂类

* createMedia 创建media
* playMedia 播放media
* addPlayListener 添加播放监听
* prepareMedia 准备media
* releaseMedia 释放media

#### MediaView
media的view

* addListener 添加监听
* playMedia 播放media
* releaseMedia 释放media

#### OnMediaListener
media的监听

* onMediaError 错误
